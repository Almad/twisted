Twisted Logging

Overview

Many people aren't aware, but Twisted provides you with a global logging facility. If you start your application using twistd, all you need to do to log messages to the logfile is

>>> from twisted.python import log >>> log.msg("I am a log message") >>>

However, Twisted's logging facility allows you to do much more. The logging module is really a global event framework, and it is possible to use it to do conditional logging or statistics gathering in large complex applications.

Log Observers: Big Brother is Watching

Log Events

The logging framework is based on events. When you do log.msg("I am a log msg"), the framework creates a dict that is sent to a list of registered observers.
